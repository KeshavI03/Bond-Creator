# Bond-Creator
A program to generate chemical bonds for simple compounds given the compounds equation
<br>
##Example:
```python
bond('HCOOH')
```
###Output:
```bash
[[0, [1, 1]], [1, [0, 1], [3, 1], [2, 2]], [2, [1, 2]], [3, [4, 1], [1, 1]], [4, [3, 1]]]
bond
      ['C']
        |
      ['H']
bond
      ['H']
        |
['O']-['C']=['O']
bond
      ['C']
        ||
      ['O']
bond
      ['H']
        |
['C']-['O']
bond
      ['O']
        |
      ['H']
```
